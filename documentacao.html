Engenharia e Qualidade de Software

1. Introdução

O ClosetLife é um sistema web desenvolvido para auxiliar usuários na organização de roupas, favoritos, categorias e gerenciamento digital do closet pessoal. O projeto foi criado utilizando conceitos de Engenharia e Qualidade de Software, priorizando organização visual, responsividade, experiência do usuário e facilidade de navegação.

Público-alvo: Pessoas que desejam organizar roupas, looks e peças de forma digital.

Escopo do Projeto: O sistema permite cadastrar roupas, favoritar peças, visualizar agenda, acessar área financeira e organizar o closet digital através de uma interface moderna e responsiva.

2. Objetivos do Sistema

  • Permitir cadastro de peças
  • Facilitar gerenciamento do closet
  • Criar interface intuitiva

19. Conclusão

O desenvolvimento do ClosetLife ajudou na aplicação dos conteúdos aprendidos em Engenharia e Qualidade de Software de forma prática e mais fácil de entender.

Durante o projeto foi possível trabalhar com planejamento, organização do sistema, documentação, modelagem UML e desenvolvimento web.

O sistema foi criado com o objetivo de facilitar a organização digital de roupas e informações do usuário, trazendo uma interface simples, moderna e responsiva.

Como melhorias futuras, o ClosetLife poderá receber sistema de login, banco de dados online e novas funcionalidades para personalização do closet.
