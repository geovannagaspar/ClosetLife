1. Introdução

O ClosetLife é uma aplicação web desenvolvida com o objetivo de auxiliar usuários no gerenciamento digital de roupas, organização do guarda-roupa e controle visual de peças pessoais. O sistema foi criado aplicando conceitos fundamentais de Engenharia e Qualidade de Software, utilizando práticas relacionadas à modelagem, arquitetura em camadas, responsividade, usabilidade e organização modular do código. A proposta do projeto consiste em oferecer uma solução simples, moderna e intuitiva para facilitar a administração de roupas, favoritos, agenda pessoal e controle financeiro relacionado ao closet.

Público-alvo: Usuários que desejam organizar roupas, montar combinações e visualizar informações do closet de forma digital e acessível.

Escopo do Projeto: O sistema permite cadastro, visualização e gerenciamento de peças de roupas, além de recursos complementares como favoritos, agenda e controle financeiro, utilizando armazenamento local e interface responsiva.

5. Arquitetura do Sistema

O ClosetLife utiliza arquitetura em camadas, separando interface visual, lógica de negócio e persistência de dados. Essa organização facilita manutenção, escalabilidade e reutilização do código, além de melhorar a estrutura geral do sistema.

Camada Tecnologia
Interface HTML5 e CSS3
Lógica JavaScript
Persistência LocalStorage

6. Paradigmas de Desenvolvimento

O sistema utiliza conceitos de Programação Orientada a Objetos (POO) para organização das estruturas e componentes do projeto, além de práticas funcionais aplicadas em funções JavaScript responsáveis pela manipulação de dados e atualização dinâmica da interface. Essa abordagem contribui para maior organização, reutilização de código e facilidade de manutenção futura.

19. Conclusão

O desenvolvimento do ClosetLife permitiu aplicar, de forma prática, os principais conceitos estudados na disciplina de Engenharia e Qualidade de Software. Durante o projeto foram trabalhados processos relacionados ao levantamento de requisitos, modelagem UML, arquitetura em camadas, organização do código, testes, documentação e publicação da aplicação.

A construção do sistema contribuiu para o desenvolvimento de conhecimentos técnicos em HTML, CSS e JavaScript, além de proporcionar maior compreensão sobre planejamento e estruturação de sistemas web responsivos.

O ClosetLife foi desenvolvido com foco em usabilidade, organização visual e praticidade, oferecendo uma solução moderna para gerenciamento digital de roupas e informações relacionadas ao closet pessoal.

Como melhorias futuras, pretende-se implementar autenticação de usuários, integração com banco de dados online, personalização avançada do sistema e novas funcionalidades voltadas para recomendações inteligentes de looks e gerenciamento completo do guarda-roupa.
