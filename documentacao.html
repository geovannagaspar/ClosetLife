Engenharia e Qualidade de Software

1. Introdução

O ClosetLife é uma aplicação web desenvolvida com o objetivo de auxiliar usuários no gerenciamento digital de roupas, organização do guarda-roupa e controle visual de peças pessoais. O sistema foi criado aplicando conceitos fundamentais de Engenharia e Qualidade de Software, utilizando práticas relacionadas à modelagem, arquitetura em camadas, responsividade, usabilidade e organização modular do código.

Público-alvo: Usuários que desejam organizar roupas e visualizar informações do closet de forma digital e acessível.

Escopo do Projeto: O sistema permite cadastro, visualização e gerenciamento de peças de roupas, além de recursos complementares como favoritos, agenda e controle financeiro.

2. Objetivos do Sistema

  • Desenvolver um sistema web funcional para gerenciamento de roupas
  • Aplicar conceitos de Engenharia e Qualidade de Software
  • Garantir usabilidade e navegação intuitiva
  • Implementar interface responsiva para múltiplos dispositivos
  • Utilizar armazenamento local para persistência dos dados

3. Requisitos Funcionais

  • RF01 - Permitir cadastro de roupas
  • RF02 - Permitir upload de imagens com compressão
  • RF03 - Favoritar peças em tempo real
  • RF04 - Remover roupas cadastradas
  • RF05 - Exibir dashboard principal
  • RF06 - Navegar entre páginas
  • RF07 - Armazenar dados no navegador (LocalStorage)

4. Design e Identidade Visual

O design do ClosetLife foi planejado para transmitir elegância e organização, utilizando uma paleta de tons neutros e pastéis.

Elemento Especificação
Cores Bege (#FDFBF8), Dourado Suave (#C8A97E), Cinza (#333)
Tipografia Montserrat (Google Fonts)
Iconografia Feather Icons

5. Arquitetura do Sistema

O ClosetLife utiliza arquitetura em camadas, separando interface visual, lógica de negócio e persistência de dados.

Camada Tecnologia
Interface HTML5 e CSS3
Lógica JavaScript (ES6+)
Persistência LocalStorage API e Canvas API

6. Qualidade e Testes

O sistema foi desenvolvido priorizando princípios de qualidade de software, incluindo usabilidade, organização modular, responsividade e desempenho. Realizamos testes de carga de imagens e validação de layout em dispositivos móveis.
