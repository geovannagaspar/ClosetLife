Engenharia e Qualidade de Software

1. Introdução

O ClosetLife é uma aplicação web desenvolvida com o objetivo de auxiliar usuários no gerenciamento digital de roupas, favoritos, agenda pessoal e organização visual do closet. O sistema foi construído aplicando conceitos de Engenharia e Qualidade de Software, priorizando usabilidade, organização estrutural, responsividade e experiência do usuário. A aplicação busca oferecer uma solução simples, moderna e acessível para o controle de peças de vestuário de forma digital.

Público-alvo: Usuários que desejam organizar roupas, looks e informações pessoais de maneira prática.

Escopo do Projeto: O sistema permite cadastrar roupas, armazenar imagens, gerenciar favoritos, organizar agenda pessoal e controlar informações financeiras através de uma interface moderna e responsiva.

6. Paradigmas de Desenvolvimento

O sistema utiliza conceitos de Programação Orientada a Objetos (POO) para melhor organização da lógica da aplicação, facilitando reutilização, manutenção e escalabilidade do código.

Também foram utilizados conceitos de programação funcional para manipulação de listas, eventos e tratamento de dados.

O desenvolvimento do ClosetLife ajudou na aplicação prática dos conteúdos estudados em Engenharia e Qualidade de Software, permitindo compreender melhor as etapas do desenvolvimento de sistemas web.

Durante o projeto foi possível trabalhar conceitos de planejamento, documentação, modelagem UML, organização estrutural e desenvolvimento de interfaces responsivas.

O sistema foi desenvolvido com o objetivo de facilitar a organização digital de roupas e informações do usuário, oferecendo uma experiência moderna, intuitiva e funcional.
