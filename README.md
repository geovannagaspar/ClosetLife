# Closet Life - Digital Closet System

Sistema web de gerenciamento de closet digital desenvolvido para a disciplina de Engenharia e Qualidade de Software.

---

# Descrição do Projeto

O ClosetLife foi criado com o objetivo de auxiliar usuários na organização de roupas, planejamento de looks e gerenciamento do guarda-roupa de forma prática, moderna e intuitiva.

O sistema permite cadastrar peças, visualizar favoritos, organizar looks em uma agenda e acompanhar estatísticas do closet através de um painel analítico.

---

# Funcionalidades

## Inventário

* Cadastro de roupas
* Organização por categorias
* Busca dinâmica de peças
* Remoção de itens

## Favoritos

* Sistema de favoritagem
* Visualização rápida de peças favoritas

## Agenda de Looks

* Planejamento de looks
* Organização por data
* Definição de mood/estilo

## Analytics

* Estatísticas do closet
* Categoria mais utilizada
* Visualização gráfica com Chart.js

## Persistência Local

* Armazenamento utilizando LocalStorage
